El objetivo en este proyecto es desarrollar habilidades de innovación y emprendimiento, ofreciendo un producto destinado a distintos usuarios. Se busca ofrecer un sistema de seguridad domiciliaria y hacia artefactos como los smartphones a un público general, y así lograr innovar con este proyecto en este ámbito. Se aplicarán diversas metodologías y técnicas de estudio para la elaboración del producto.
